# Brun's Constant

## Description of constant

$C\_{81a}$, Brun's Constant, is the sum of the reciprocals of the twin primes.

## Known upper bounds

| Bound                  | Reference      | Comments                                      |
| ---------------------- | -------------- | --------------------------------------------- |
| 2.175398               | [K2007]        |                                               |
| 2.288513               | [PT2018]       |                                               |


## Known lower bounds

| Bound                  | Reference      | Comments                                      |
| ---------------------- | -------------- | --------------------------------------------- |

## Additional comments

Similar constants exist for other prime families, e.g., cousin primes.
