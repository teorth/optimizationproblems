# Flatness constant in dimension 3

## Description of constant

A convex body $K\subset\mathbb{R}^d$ is hollow (lattice-free) with respect to a lattice $\Lambda$ if $\operatorname{int}(K)\cap\Lambda=\varnothing$.
<a href="#CS2019-hollow-def">[CS2019-hollow-def]</a>

For a hollow body, the lattice width is

$$
w(K)\ :=\ \min_{u\in\mathbb{Z}^d\setminus\{0\}}
\left(\max_{x\in K}u\cdot x-\min_{x\in K}u\cdot x\right).
$$

<a href="#ACMS2021-width-def">[ACMS2021-width-def]</a>

The flatness constant in dimension $d$ is

$$
\mathrm{Flt}(d)\ :=\ \sup\{w(K): K\subset\mathbb{R}^d\ \text{hollow convex body}\},
$$

and is finite for each fixed $d$.
<a href="#ACMS2021-Flt-def">[ACMS2021-Flt-def]</a>
<a href="#CS2019-flatness-def">[CS2019-flatness-def]</a>

We define

$$
C_{73}\ :=\ \mathrm{Flt}(3).
$$

An explicit hollow tetrahedron gives

$$
\mathrm{Flt}(3)\ \ge\ 2+\sqrt{2}.
$$

<a href="#CS2019-thm-1-1">[CS2019-thm-1-1]</a>

An explicit published upper bound is

$$
\mathrm{Flt}(3)\ <\ 3.972.
$$

<a href="#ACMS2021-ub-3-972">[ACMS2021-ub-3-972]</a>

Hence the best established range is

$$
2+\sqrt{2}\ \le\ C_{73}\ <\ 3.972.
$$
