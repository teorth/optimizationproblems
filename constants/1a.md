# An autocorrelation constant related to Sidon sets

## Description of constant

$C\_{1a}$ is the largest constant for which one has
$$
 \max\_{-1/2 \leq t \leq 1/2} \int\_{\mathbb{R}} f(t-x) f(x)\ dx \geq C\_{1a} \left(\int\_{-1/4}^{1/4} f(x)\ dx\right)^2
$$
for all non-negative $f \colon \mathbb{R} \to \mathbb{R}$.
