# Union closed sets conjecture

## Description of constant

$C\_{6}$ is the largest constant such that any union-closed family of sets on an $N$-element ground set has an element contained in at least $C\_{6}$ fraction of the sets.

## Known upper bounds

| Bound | Reference | Comments |
| ----- | --------- | -------- |
| $1/2 = 0.5$ | [F1995] | Conjectured (in 1976) to be optimal |

## Known lower bounds

| Bound | Reference | Comments |
| ----- | --------- | -------- |
| $0$ | Trivial | |
| $0.1$ | [G2022] | |
| $(3-\sqrt{5})/2 = 0.381966\dots$ | [AHS2022], [CL2022], [P2022], [S2022] |
| $>(3-\sqrt{5})/2$ | [S2022] |
| $0.38234$ | [C2022], [Y2022] |
| $0.38271$ | [L2023] |
