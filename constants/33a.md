# Ihara constant over $\mathbf{F}_2$

## Description of constant

$C_{33}=A(2)$ is the **Ihara constant** over $\mathbb{F}_2$.
<a href="#DM2013-def-Aq">[DM2013-def-Aq]</a>

For each integer $g\ge 1$, let

$$
N_{2}(g)
:=
\max\bigl\{\#X(\mathbb{F}_2)\;:\; X/\mathbb{F}_2 \text{ a smooth projective geometrically integral curve of genus } g\bigr\}.
$$

<a href="#DM2013-def-Nqg">[DM2013-def-Nqg]</a>

Then
$$
A(2)
:=
\limsup_{g\to\infty}\frac{N_{2}(g)}{g}.
$$

<a href="#DM2013-def-Aq">[DM2013-def-Aq]</a>

## Known upper bounds

| Bound | Reference | Comments |
| ----- | --------- | -------- |
| $2\sqrt2 \approx 2.82843$ | Classical (Weil bound) | From $\#X(\mathbb{F}_2)\le 2+1+2g\sqrt2$, hence $\frac{N_2(g)}{g}\le 2\sqrt2+\frac{3}{g}$. <a href="#DM2013-weil-bound">[DM2013-weil-bound]</a> |
| $\sqrt2-1 \approx 0.41421$ | [DV1983](#DV1983) | Drinfeld--Vlăduţ (Ihara) bound: $A(q)\le \sqrt q-1$ for every prime power $q$. <a href="#DM2013-dv-bound">[DM2013-dv-bound]</a> |

## Known lower bounds

| Bound | Reference | Comments |
| ----- | --------- | -------- |
| $0$ | Trivial | Since $N_2(g)\ge 0$. |
| $2/9 \approx 0.22222$ | [Ser1983](#Ser1983), [Sch1992](#Sch1992) | Serre's class field tower method; explicit constructions over $\mathbb{F}_2$. <a href="#Bee2022-lb-2-9">[Bee2022-lb-2-9]</a> |
| $81/317 \approx 0.25552$ | [NX1998](#NX1998) | Improves the $2/9$ bound. <a href="#Bee2022-lb-81-317">[Bee2022-lb-81-317]</a> |
| $97/376 \approx 0.25798$ | [XY2007](#XY2007) | Improves the $81/317$ bound. <a href="#DM2013-prop1.1">[DM2013-prop1.1]</a> |
| $39/129 \approx 0.30233$ | [DM2013](#DM2013) | Lower bound reported in [DM2013](#DM2013) (attributed there to Kuhnt's thesis). <a href="#DM2013-prop1.2">[DM2013-prop1.2]</a> |
| $0.316999\ldots$ | [DM2013](#DM2013) | Lower bound from [DM2013](#DM2013) (also listed in [Bee2022](#Bee2022)). <a href="#DM2013-thm1.1">[DM2013-thm1.1]</a> <a href="#Bee2022-lb-best">[Bee2022-lb-best]</a> |

## Additional comments and links

- **Status.** The exact value of $A(2)$ is unknown. As of the survey [Bee2022](#Bee2022), one has
  $$
  0.316999\ldots \ \le\ A(2)\ \le\ \sqrt2-1\approx 0.41421.
  $$
  <a href="#Bee2022-nonsquare-open">[Bee2022-nonsquare-open]</a> <a href="#DM2013-thm1.1">[DM2013-thm1.1]</a> <a href="#DM2013-dv-bound">[DM2013-dv-bound]</a>

- **Square vs. non-square fields.** For $q$ a square, the Drinfeld--Vlăduţ upper bound is sharp:
  $A(q)=\sqrt q-1$, via explicit towers of function fields (e.g. the Garcia--Stichtenoth tower) [GS1995](#GS1995).
  For non-square $q$ (in particular $q=2$), no exact value is known.
  <a href="#DM2013-square-q">[DM2013-square-q]</a> <a href="#Bee2022-gs-tower">[Bee2022-gs-tower]</a> <a href="#Bee2022-nonsquare-open">[Bee2022-nonsquare-open]</a>

- **Connection to coding theory (TVZ bound).** The quantity $A(q)$ controls the asymptotic performance of algebraic-geometry codes.
  In particular, the Tsfasman--Vlăduţ--Zink bound expresses an asymptotic rate--distance tradeoff in terms of $A(q)$; see [TVZ1982](#TVZ1982), [Sti2009](#Sti2009).
  <a href="#Sti2005-TVZ">[Sti2005-TVZ]</a> <a href="#Bee2022-coding-theory">[Bee2022-coding-theory]</a>

- **Tables for fixed genus.** For small genera, records and exact maxima for $N_2(g)$ are tabulated at [manypoints.org](https://manypoints.org/).
  <a href="#Fab2022-manypoints">[Fab2022-manypoints]</a>

- Surveys and background: [Bee2022](#Bee2022), [Sti2009](#Sti2009).
